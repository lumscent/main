# Product Design Knowledge

Product Design Knowledge — это репозиторий продуктового и UX-контекста для команды продуктовых дизайнеров.

Он используется для хранения:
- информации о продуктах;
- пользовательских ролей и сценариев;
- исследовательских выводов;
- UX/UI-паттернов;
- принятых дизайн-решений;
- правил дизайн-ревью;
- шаблонов для постановки и описания задач;
- инструкций для AI-инструментов.

Главная цель репозитория — помочь дизайнерам принимать решения на основе актуального продуктового контекста, исследований и уже принятых договорённостей, а не каждый раз начинать с чистого листа.

---

## Для кого этот репозиторий

Репозиторий предназначен для:

- продуктовых дизайнеров;
- lead product designer;
- UX/UI-дизайнеров;
- исследователей;
- продуктовых менеджеров;
- AI-инструментов, которые помогают в дизайн-задачах.

---

## Что можно делать с помощью репозитория

Репозиторий помогает:

- быстрее погружаться в продукт;
- формулировать интерфейсные задачи;
- готовиться к дизайн-ревью;
- проверять решения на соответствие UX-принципам;
- использовать прошлые исследования в новых задачах;
- фиксировать принятые дизайн-решения;
- сохранять единый подход к UX/UI-паттернам;
- давать Claude или другому AI-инструменту понятный рабочий контекст.
